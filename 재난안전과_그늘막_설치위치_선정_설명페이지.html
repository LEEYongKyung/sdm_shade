운영 웹페이지 열기

그늘막 설치 후보지 선정,
이제 데이터로 설명합니다.

기존에는 민원 위치를 중심으로 후보지를 검토하면서 약 7일이 소요되었습니다. 본 시스템은 횡단보도, 인도 폭, 도로, 교차로, 고령자, 무더위쉼터, 기존 그늘막 정보를 한 화면에서 비교하여 객관적인 설치 적합 위치를 찾는 업무용 웹페이지입니다.

7일 기존 후보 검토 소요
451개 기본 횡단보도 후보
20점 설치 필요도 평가
재난안전과 설명용 웹페이지 안내자료

1. 기존 업무에서 어려웠던 점

후보지를 고르는 기준이 명확하게 보이지 않으면, 설치 우선순위와 예산 투입 근거를 설명하기 어렵습니다.

후보 검토에 시간이 오래 걸림

설치 후보지를 찾기 위해 여러 자료를 따로 확인해야 했고, 후보 검토에 약 일주일이 소요되었습니다.

민원 중심 검토의 한계

민원이 들어온 위치를 우선 살피다 보니, 실제로 어느 위치가 더 필요한지 객관적으로 비교하기 어려웠습니다.

현장 부적합 가능성

인도 폭, 기존 그늘막과의 중복, 주변 시설물 등 설치 불가 요소가 현장에서 뒤늦게 확인될 수 있었습니다.

2. 시스템이 해결하는 방식

사람이 일일이 후보지를 찾는 방식에서, 데이터가 먼저 후보를 분류하고 담당자가 현장을 최종 확인하는 방식으로 바꿉니다.

후보 생성

서울시 대로변 횡단보도 위치를 기본 후보지로 사용합니다.

필수 조건 확인

인도 폭 3.5m 이상, 기존 그늘막 80m 이내 여부를 먼저 확인합니다.

가점 계산

도로, 교차로, 고령자, 쉼터 접근성을 기준으로 점수를 계산합니다.

지도 검토

선정 후보, 현장 확인 후보, 제외 후보를 지도와 표에서 동시에 확인합니다.

업무 활용

후보 리스트를 엑셀로 내려받고, 매년 설치된 위치를 다시 업로드합니다.

3. 웹페이지에서 볼 수 있는 화면

지도 위에 평가 근거가 함께 보이므로, 특정 위치가 왜 선정 후보인지 한눈에 설명할 수 있습니다.

후보 현황 요약 원천 후보, 선정, 현장 확인, 제외 건수를 상단에서 바로 확인합니다.
지도 기반 근거 확인 고령자 분포, 기존 그늘막, 무더위쉼터를 지도 위에 함께 표시합니다.
점수체계 조정 오른쪽 점수체계 항목을 켜고 끄며 평가 결과를 다시 비교합니다.
엑셀·업로드 연계 후보 리스트 다운로드와 연도별 설치 위치 업로드를 지원합니다.

4. 평가 기준

총점은 20점이며, 필수 설치 조건은 점수에 포함하지 않고 미충족 시 후보에서 제외합니다.

구분 평가 항목 판단 기준 점수
필수 인도 폭 3.5m 이상 그늘막 설치 기준상 최소 인도 폭을 충족해야 합니다. 필수
가점 도로/간선도로성 도로명주소 도로구간과 실폭도로 자료를 기준으로 도로 폭과 주도로 여부를 평가합니다. 0~5점
가점 교차로/사거리 인접 교차로와 가까울수록 보행 대기 수요가 높다고 보고 점수를 부여합니다. 0~5점
가점 고령자 밀집 지역 법정동 기준 65세 이상 인구 비율이 높은 지역에 높은 점수를 부여합니다. 0~6점
가점 무더위쉼터 접근 부족 가까운 무더위쉼터가 멀수록 그늘막 설치 필요성이 높다고 평가합니다. 0~4점
필수 기존 그늘막 80m 이내 제외 중복 설치를 막기 위해 기존 그늘막과 가까운 후보는 제외합니다. 필수

공공데이터, 브이월드 공간자료, 서대문구 내부자료를 통합하여 후보지 평가에 사용합니다.

6. 운영 방식

직원은 웹브라우저로 접속하고, 자료는 서버와 PostgreSQL 데이터베이스에서 관리합니다.

웹 접속

별도 프로그램 설치 없이 웹브라우저에서 후보지 지도와 리스트를 확인합니다.

점수체계 조정

평가 항목 체크박스를 켜고 끄며 조건별 시뮬레이션 결과를 비교합니다.

엑셀 활용

선정 후보, 제외 후보, 기존 그늘막 목록을 엑셀로 내려받아 보고자료로 활용합니다.

연도별 갱신

새로 설치한 그늘막 위치를 업로드하면 다음 평가에서 기존 그늘막으로 반영됩니다.

7. 도입 기대효과

업무 시간 단축뿐 아니라, 왜 해당 위치가 필요한지 설명할 수 있는 근거가 생깁니다.

시간 단축

후보지 검토 업무 효율화

여러 자료를 따로 확인하던 과정을 한 화면에서 처리합니다.

객관성

민원 중심에서 기준 중심으로

인도 폭, 도로, 교차로, 고령자, 쉼터, 기존 시설을 함께 반영합니다.

설명력

선정 근거 시각화

지도와 점수표를 통해 후보 선정 이유를 부서 내외부에 설명할 수 있습니다.

현장성

현장 확인 대상 선별

자동 제외 후보와 현장 확인 후보를 구분해 현장 방문 우선순위를 정할 수 있습니다.

지속성

매년 반복 사용

설치 완료 위치를 계속 누적하여 다음 연도 평가에 반영합니다.

공유

회의·보고자료 연계

엑셀 다운로드와 웹 화면을 이용해 검토 결과를 쉽게 공유합니다.

8. 향후 고도화 방향

현재 v1.0은 위치 선정과 자료 확인 중심이며, 다음 단계에서는 현장 검토 관리 기능을 추가할 수 있습니다.

현장검토 관리

후보별 현장 확인 결과, 설치 가능 여부, 불가 사유, 사진, 메모를 저장합니다.

관리자 로그인

누가 데이터를 수정했는지 이력을 남기고, 관리 기능은 권한 있는 사용자만 접근하도록 합니다.

데이터 갱신 자동화

공공데이터와 내부자료 갱신 절차를 정례화해 매년 같은 기준으로 평가합니다.

핵심은 “빠르게 고르는 것”보다
“설명 가능한 기준으로 고르는 것”입니다.

이 시스템은 재난안전과가 그늘막 설치 후보지를 검토할 때 필요한 자료를 한곳에 모으고, 설치 필요도와 제외 사유를 함께 보여주는 의사결정 지원 도구입니다.
