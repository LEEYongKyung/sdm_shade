# 서대문구 고령인구 + 횡단보도/그늘막 + 그늘막 후보지 TOP100 (Leaflet)

## 포함 기능
- 행정동(서대문구) 고령인구 코로플레스
- 🚸 횡단보도 / 🌳 기존 그늘막: 클러스터(분리 카운트)
- ⭐ 후보지 TOP100: 별 아이콘 + 좌측 리스트(점수순)
  - 리스트 클릭 시 해당 위치로 이동(줌/센터) + 팝업 오픈
  - 검색(동/노드ID) 가능

## 실행 방법
### VSCode Live Server (추천)
- 폴더 열기 → `index.html` 우클릭 → Open with Live Server

### Python 서버
- Windows: `py -m http.server 8000`
- Mac/Linux: `python3 -m http.server 8000`
- `http://localhost:8000`


## 변경사항
- 🚸 횡단보도 / 🌳 그늘막이 서로 **섞여서 한 클러스터로 표시되지 않도록** 분리했습니다.
- 레이어 토글도 각각 따로 제공합니다.


## 추가 기능
- 🚸/🌳 클러스터(묶음) 숫자 배지를 파랑/초록 색상으로 구분했습니다.
- 좌측 상단에 `CSV 다운로드` 버튼을 추가했습니다.
- 다운로드 CSV 컬럼: 순위/행정동/노드ID/총점+항목별/경도/위도
